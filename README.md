Sentilytics is a comprehensive sentiment analysis platform developed within the **Department of Computer Science at the University of Layyah**. It transforms raw social media data into actionable insights using advanced Natural Language Processing (NLP).

## üåü Key Features

- **Real-Time Analysis**: Monitor live streams of social media sentiment.
- **Dual Platforms**: Choice between a high-performance **FastAPI** dashboard and an interactive **Streamlit** data application.
- **Thematic Interface**: Support for **Light and Dark Modes** with smooth transitions.
- **KPI Tracking**: Real-time metrics for sentiment volume, engagement, and crisis levels.
- **Interactive Visualizations**: Dynamic pie charts, trend lines, and keyword frequency bars.
- **University Branding**: Integrated branding for the University of Layyah.

## üöÄ Getting Started

### Prerequisites

- Python 3.10+
- Chrome Browser (Recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sentilytics.git
   cd sentilytics
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Platforms

#### 1. FastAPI Dashboard (Full Version)
```bash
python sentilytics/run.py
```
Access at:[https://sentilyticstech.vercel.app/]

#### 2. Streamlit Dashboard (Interactive Version)
```bash
cd sentilytics-app
streamlit run app.py
```
Access at: [http://localhost:8501](http://localhost:8501)
